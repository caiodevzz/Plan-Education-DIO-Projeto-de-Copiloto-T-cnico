# Plan-Education-DIO-Projeto-de-Copiloto-T-cnico
Repositório do projeto **Plan Education DIO**, focado no uso de IA como copiloto técnico em modo PLAN. A proposta é planejar implementações antes de escrever código, definindo escopo, etapas, riscos e validações. O projeto utiliza Node.js e TypeScript para explorar boas práticas de arquitetura e desenvolvimento orientado a planejamento.
